# JoshStatusBarHUD
A simple useful status bar HUD framework

##显示成功信息
```objc
 [JoshStatusBarHUD showSuccess:@"成功"];
```

##显示失败信息
```objc
  [JoshStatusBarHUD showError:@"失败"];
```

##显示正在加载信息
```objc
 [JoshStatusBarHUD showLoading:@"正在加载中"];
```

##显示文本信息
```objc
 [JoshStatusBarHUD showMessage:@"没有图片哦"];
```

##隐藏HUD
```objc
 [JoshStatusBarHUD hide];
```
